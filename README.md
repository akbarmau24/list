#Aplikasi To-Do List

##Deksripsi
Aplikasi Sederhana berbasis PHP untuk mencatat tugas harian.

##Fitur
-Tambah tugas
-Tandai tugas selesai
-Hapus Tugas

##Struktur Folder
-'index.php'- halaman utama
-'style.css'- file css

##Cara Menjalankan
1. Salin folder ke 'htdocs/'
2. Jalankan XAMPP dan bukan di 'http://localhost/list/index.php'
